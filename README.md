# Student-Performance-Analysis
A data science project focused on data preprocessing, visualization, and storytelling using student performance data.

## Project Overview
This project focuses on data preprocessing, visualization, and storytelling using student performance data.

## Dataset Information
- Total Rows: 395
- Total Columns: 31
- Target Variable: passed

## Data Preprocessing
- Loaded dataset using Pandas
- Checked missing values
- Checked duplicate records
- Verified data consistency

## Data Visualization
1. Passed vs Failed Students
2. Gender Distribution
3. Study Time vs Result
4. Absences Distribution

## Key Insights (Storytelling)

### Insight 1
Most students successfully passed the examination.

### Insight 2
Students who spend more time studying generally achieve better results.

### Insight 3
Higher absenteeism may negatively impact academic performance.

### Insight 4
The dataset contains a balanced representation of male and female students.

## Conclusion
Data preprocessing ensured data quality, while visualizations helped identify important factors affecting student performance. Study time showed a positive impact on results, whereas excessive absences may reduce academic success.
